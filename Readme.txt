Maybot
------

Maybot is a skill for Amazon Alexa that completes sentences in the style of Theresa May's
parliamentary spoken contributions. The server can be run either on a user's local computer
or on a cloud server, in AWS or elsewhere.

Installation
------------

1. sudo pip install flask-ask
2. Download ngrok from https://ngrok.com/download and unzip
3. ./run.sh
4. In Alexa Skills Kit console https://developer.amazon.com/edw/home.html set skill
   Configuration -> Endpoint to ngrok https URL
